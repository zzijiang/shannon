# Shannon · 香农学习平台

## 产品定位
面向初中生的 Python + 数据科学自学平台，对标 NOAI 水平测试。核心目标：**让孩子能用、乐意用**。

## 目标用户
- 中国初中生（12-15 岁）
- 零基础到有一定编程经验
- 自学场景，无老师实时指导

## 核心设计原则

### 1. 行为优先，内容其次
先设计"每天 10 分钟的行为循环"，再往里填内容。频率 > 时长，间隔重复碾压集中突击。

### 2. 三种教学模式交替使用
- **刷卡模式（Drill）**：30秒-2分钟/题，语法记忆、API 辨析。驱动每日回访。
- **引导探索模式（Guided Exploration）**：5-8 页交互微叙事，每页一个动作，Brilliant 式。用于新概念首次学习。
- **项目模式（Project）**：15-30 分钟脚手架式项目，每步可运行、有可视产出。用于综合应用。

### 3. 一个按钮原则
孩子面前永远只有"继续"。选择瘫痪是自学失败的首要原因，默认路径不需要决策。

### 4. 损失厌恶驱动留存
连胜 > 积分累积。每日目标 + 周匿名榜 + streak 机制。不用"0/60"这种大山式进度展示。

### 5. 产出可炫耀
每 3-5 个单元穿插产出项目，结果必须可视、可截图、可分享。社交货币是青少年产品最被低估的留存杠杆。

### 6. AI 助教是陪伴者，不是答案机
永远不输出完整可运行代码。苏格拉底式反问 + 分级 hint。答对时也要出现（正向锚点），不只在卡住时出现。

### 7. 双层叙事
孩子看到的：用 Python 做酷东西。家长看到的：NOAI 考纲覆盖、学习报告。两层分开，不要混在一起。

## 技术栈
- 前端：HTML/CSS/JS 单页应用
- 浏览器端 Python：Pyodide（WebAssembly）
- 图表：Chart.js（探索/练习）+ matplotlib via Pyodide（项目实战）
- 中文字体：LXGW WenKai（通过 CDN 加载到 Pyodide）
